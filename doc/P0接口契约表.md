# P0 接口契约表

> 本文档定义 P0 最小闭环批次所需的全部 REST API 接口契约，供前后端联调与代码生成使用。
> 来源：`产品需求文档 v1.0.md § 1.4 P0 + § 6 API 设计约定`
>
> **统一约定**：
> - 基础路径：`/api/v1`
> - 响应格式：`{ "code": 0, "message": "success", "data": {...}, "request_id": "uuid" }`
> - 分页参数：`page`, `page_size`（最大 100），返回 `total`, `list`
> - 鉴权：`Authorization: Bearer <token>`（除登录/注册/健康检查外）
> - 敏感操作：先调用 `POST /api/v1/auth/step-up` 获取 `step_up_token`，再通过请求头 `X-UD-Step-Up-Token: <step_up_token>` 提交；一次性敏感操作的令牌仅可使用 1 次。
> - 错误码号段：认证 10000-10999 | 工单 11000-11999 | 业务域 15000-15999 | 权限 16000-16999 | 通知 14000-14999 | 附件 17000-17999
> - 业务域可见策略编码：`visibility_policy_codes` 为数组，取值 `public` / `domain_customer_only` / `channel_only`；`public` 与其他两项互斥，`domain_customer_only` 与 `channel_only` 可组合。

## 1. 认证与账号（10000-10999）

### 1.1 登录

| 项 | 值 |
|---|---|
| **Method** | `POST` |
| **Path** | `/api/v1/auth/login` |
| **权限** | 无（公开） |
| **Request Body** | `{ "login_name": string, "password": string, "portal_type": "customer" \| "staff", "captcha_token"?: string }` |
| **Response** | `{ "access_token": string, "refresh_token": string, "expires_in": number, "portal_type": string, "subject_id": string }` |
| **错误码** | `10001` 用户名或密码错误 · `10002` 账号已锁定 · `10003` 验证码校验失败 · `10004` 账号已禁用 |

### 1.2 刷新 Token

| 项 | 值 |
|---|---|
| **Method** | `POST` |
| **Path** | `/api/v1/auth/refresh` |
| **权限** | 无（携带 refresh_token） |
| **Request Body** | `{ "refresh_token": string }` |
| **Response** | `{ "access_token": string, "refresh_token": string, "expires_in": number }` |
| **错误码** | `10010` Refresh Token 无效或已过期 · `10011` 会话已被踢除 |

### 1.3 登出

| 项 | 值 |
|---|---|
| **Method** | `POST` |
| **Path** | `/api/v1/auth/logout` |
| **权限** | 已认证 |
| **Request Body** | `{ "refresh_token"?: string }` |
| **Response** | `{}` |

### 1.4 获取当前用户信息

| 项 | 值 |
|---|---|
| **Method** | `GET` |
| **Path** | `/api/v1/auth/me` |
| **权限** | 已认证 |
| **Response** | `{ "subject_id": string, "account_id": string, "account_type": "staff" \| "customer", "login_name": string, "display_name": string, "avatar_url"?: string, "phone"?: string, "email"?: string, "platform_roles": string[], "current_domain"?: { "id": string, "name": string, "roles": string[], "permissions": string[] } }` |

### 1.5 客户注册

| 项 | 值 |
|---|---|
| **Method** | `POST` |
| **Path** | `/api/v1/auth/register` |
| **权限** | 无（公开） |
| **Request Body** | `{ "login_name": string, "password": string, "display_name": string, "phone": string, "email"?: string, "domain_id"?: string, "invitation_code"?: string, "captcha_token": string }` |
| **Response** | `{ "access_token": string, "refresh_token": string, "account_id": string }` |
| **错误码** | `10020` 用户名已存在 · `10021` 手机号已注册 · `10022` 邀请码无效 · `10023` 该域不允许自注册 |

### 1.6 找回密码

| 项 | 值 |
|---|---|
| **Method** | `POST` |
| **Path** | `/api/v1/auth/password/reset-request` |
| **权限** | 无（公开） |
| **Request Body** | `{ "login_name": string, "portal_type": "customer" \| "staff", "captcha_token": string }` |
| **Response** | `{ "channel": "email" \| "admin", "hint": string }` |
| **错误码** | `10030` 账号不存在 · `10031` 每日重置次数已达上限 |

### 1.7 验证重置码并修改密码

| 项 | 值 |
|---|---|
| **Method** | `POST` |
| **Path** | `/api/v1/auth/password/reset` |
| **权限** | 无（公开） |
| **Request Body** | `{ "token": string, "new_password": string }` |
| **Response** | `{}` |
| **错误码** | `10032` 重置码无效或已过期 |

### 1.8 修改密码（已登录）

| 项 | 值 |
|---|---|
| **Method** | `PUT` |
| **Path** | `/api/v1/auth/password` |
| **权限** | 已认证 |
| **Request Body** | `{ "old_password": string, "new_password": string }` |
| **Response** | `{}` |
| **错误码** | `10040` 原密码错误 · `10041` 新密码不满足复杂度 |

### 1.9 敏感操作二次验证

| 项 | 值 |
|---|---|
| **Method** | `POST` |
| **Path** | `/api/v1/auth/step-up` |
| **权限** | 已认证员工账号 |
| **Request Body** | `{ "password": string, "operation_code": string }` |
| **Response** | `{ "step_up_token": string, "expires_in": 900, "reuse_policy": "session_15m" \| "one_time", "operation_code": string }` |
| **错误码** | `10050` 需要二次验证 · `10051` 二次验证令牌无效或已过期 · `10052` 密码错误 |

> `operation_code` 由后端按敏感操作清单校验。删除业务域、身份归并、彻底删除员工/客户、全量导出、全局安全策略修改使用 `one_time`；平台 `platform_admin` 授权/回收、普通角色授权、域级 `super_admin` 授予/回收等使用 `session_15m`。

> 业务域管理员创建和管理邀请码，用于客户入域。

| Method | Path | 权限 |
|--------|------|------|
| `GET` | `/api/v1/admin/domains/{domain_id}/invitation-codes` | `domain:user_manage` |
| `POST` | `/api/v1/admin/domains/{domain_id}/invitation-codes` | `domain:user_manage` |
| `DELETE` | `/api/v1/admin/domains/{domain_id}/invitation-codes/{code_id}` | `domain:user_manage` |

**创建请求体**：`{ "channel"?: string, "expires_at"?: string, "max_uses"?: number }`

**创建响应**：`{ "id": string, "code": string }`

**错误码**：`15010` 邀请码数量超限 · `15011` 当前注册策略不支持邀请码

> `registration_policy` 为 `admin_only` 时邀请码功能不可用，创建接口直接返回 `15011`。

## 9. 附件

### 9.1 获取预签名上传凭证（对象存储模式）

| 项 | 值 |
|---|---|
| **Method** | `POST` |
| **Path** | `/api/v1/attachments/presign` |
| **权限** | 已认证 |
| **Request Body** | `{ "file_name": string, "mime_type": string, "file_size": number, "target_type": "ticket" \| "consultation" \| "knowledge", "domain_id": string }` |
| **Response** | `{ "attachment_id": string, "upload_url": string, "expires_in": number }` |
| **错误码** | `17001` 文件类型不在白名单 · `17002` 文件大小超限 · `17003` 总附件大小超限 |

> 当 `storage_type=object_storage` 且 MinIO 可用时使用本接口；若 Sprint 0 未确认 MinIO，启用 9.2 本地文件上传。

### 9.2 本地文件上传（仅本地存储降级模式）

| 项 | 值 |
|---|---|
| **Method** | `POST` |
| **Path** | `/api/v1/attachments/upload` |
| **权限** | 已认证 |
| **Content-Type** | `multipart/form-data` |
| **Form Data** | `file`, `target_type`, `domain_id` |
| **Response** | `{ "attachment_id": string, "download_url": string, "storage_type": "local" }` |
| **错误码** | `17001` 文件类型不在白名单 · `17002` 文件大小超限 · `17003` 总附件大小超限 |

> 本地存储根目录由服务端配置 `storage.local_root`，接口不得返回服务器绝对路径。`download_url` 必须是后端鉴权下载地址或短时签名 URL。

### 9.3 确认上传完成

| 项 | 值 |
|---|---|
| **Method** | `PUT` |
| **Path** | `/api/v1/attachments/{attachment_id}/confirm` |
| **权限** | 已认证 |
| **Response** | `{}` |

> 对象存储模式上传完成后调用确认接口；本地存储模式上传接口直接完成落库，可不调用确认接口。

### 9.4 获取附件访问地址

| 项 | 值 |
|---|---|
| **Method** | `GET` |
| **Path** | `/api/v1/attachments/{attachment_id}/download` |
| **权限** | 已认证且具备目标业务对象访问权 |
| **Response** | `{ "download_url": string, "expires_in": number, "storage_type": "object_storage" \| "local" }` |
| **错误码** | `17004` 附件不存在 · `17005` 无附件访问权限 |

> 对象存储模式返回对象存储短时签名 URL；本地存储模式返回后端鉴权下载 URL。附件迁移到对象存储后保持 `attachment_id` 和业务引用不变，仅更新 `storage_type`、`storage_key`、校验和与访问 URL 生成方式。本地 `storage.local_root` 必须纳入备份；清理任务只能删除无 `attachment_ref` 业务引用且超过保留期的临时/孤儿文件。
