# 文档总览

本目录用于维护 UnionDesk 项目群的产品与架构文档。

## 仓库定位
- `UnionDesk`：后端服务项目。
- `UnionDeskWeb`：前端项目（客户端 + 管理端）。

## 当前文档
- `PRD.md`：产品需求文档，定义目标、范围、角色、MVP 验收标准。
- `系统架构设计.md`：系统架构文档，定义边界、模块、部署与演进。
- `数据库设计.md`：数据库设计说明，包含实体、索引、归档与迁移约定。
- `schema.sql`：MySQL 8 初始化 DDL（含角色种子数据）。
- `技术栈方案.md`：前后端、数据库、日志、部署选型基线。

## 建议后续补充
1. 接口规范文档（OpenAPI、错误码、幂等策略）。
2. 测试与发布手册（测试分层、灰度、回滚）。
3. 运维手册（监控面板、告警、备份与恢复演练）。

## 维护规则
- 文档更新需同步版本号和日期。
- 需求边界变化需同时更新 PRD 与架构文档。
- 已上线功能需补充验收记录或变更记录。
