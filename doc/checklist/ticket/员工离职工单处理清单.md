# 员工离职工单处理清单

## 业务域
- 员工管理 / 工单

## 目标
- 明确员工进入离职池时，其在途工单的默认处理策略，避免离职流程与工单归属冲突。

## 开发任务
- [x] 在员工离职池说明中明确：未关闭工单自动解除指派并回退至未分配队列。
- [x] 明确离职处理需保留历史指派记录，并通知域管理员接管。
- [x] 在域级离职池说明中同步同样的未关闭工单处理口径。
- [x] 复查离职流程口径，确认不再要求阻止离职或强制转派给单一管理员。

## 验证记录
- [x] 员工离职后在途工单不会丢失。
- [x] 工单仍可由域管理员重新分配。
- [x] 域内移除用户时，相关域内在途工单仍可被回退并再分配。
