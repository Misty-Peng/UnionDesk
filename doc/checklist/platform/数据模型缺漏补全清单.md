# 数据模型缺漏补全清单

## 0. 说明
- [x] 本清单用于收口功能需求中提到但第 5 节尚需补齐的数据模型实体。

## 1. 已补充实体
- [x] `domain_template`：业务域复制配置与提炼模板。
- [x] `file_attachment`：统一附件主表。
- [x] `attachment_ref`：附件与工单 / 咨询 / 知识的关联表。
- [x] `login_log`：登录日志。
- [x] `system_config`：平台级系统配置。
- [x] `quick_reply_template`：客服快捷回复模板。
- [x] `staff_presence`：客服在线 / 忙碌 / 离线状态。
- [x] `domain_config`：业务域通用配置。

## 2. 已确认复用实体
- [x] `notification_template` 已在通知中心数据模型中存在。
- [x] `satisfaction_rating` 已在满意度评价数据模型中存在。

## 3. 验收
- [x] 业务域模板可被提炼和再次用于新建业务域。
- [x] 附件统一通过主表 + 关联表管理，不在工单 / 咨询 / 知识中重复建表。
- [x] 登录日志可按账号、IP、时间、结果查询。
- [x] 平台级与域级配置可分别落到对应配置表。
- [x] 客服快捷回复与在线态均有独立数据模型支撑。
