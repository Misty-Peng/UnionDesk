# 边界场景清单

## 已完成

- [x] 业务域至少保留 1 个 `domain_admin`
- [x] 系统始终至少保留 1 个激活的 `super_admin`
- [x] 被合并工单不生成待评价记录，仅主单关闭后触发评价
- [x] SLA 最高优先级时自动跳过调级动作
- [x] 客户多个待评价邀请独立展示，不做聚合
- [x] 被引用知识条目删除 / 下线后显示“引用内容已不可用”占位提示

## 验证

- 最后一位域管理员不可自行降级或移除
- 最后一位超级管理员不可被禁用或移除
- 工单合并后仅主单触发满意度评价
- SLA 违约动作在最高优先级时不会继续调级
- 通知中心可独立展示多条待评价邀请
- 引用方文档可继续展示，不因子引用失效而整体报错
