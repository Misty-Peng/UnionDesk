## 关键决策

1. **咨询模块（WebSocket）推迟到 P1**
   - P0 只保证工单闭环，咨询为增强功能
   - 先预留接口，不实现

2. **批量操作简化**
   - 批量领取/指派：前端循环调用降级
   - 导出：先支持同步 CSV，异步队列后置

3. **知识库 P2 再启动**
   - 公开知识库帮助中心属于 P2 范围
   - P0/P1 阶段不投入
